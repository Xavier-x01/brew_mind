# Work Journal

Add a dated entry to `docs/operator-daily-log.md` for today's session.

Steps:
1. Read `docs/operator-daily-log.md` (create it if missing).
2. Ask the user: "What did you work on today? Any blockers? What's your intent for tomorrow?" — accept short answers.
3. Append a new entry at the TOP of the file in this format:

```
## YYYY-MM-DD

**Did:** <what was accomplished>
**Blocker:** <one blocker or "none">
**Tomorrow:** <one intent>
```

4. Save the file and confirm the entry was added.

Keep entries honest and short — one line per field is enough.
